# TB Early Risk Detection System

A web-based system to identify early tuberculosis (TB) risk using symptom-based analysis and machine learning.

## Features
- Symptom-based TB risk assessment
- Risk classification (Low / Medium / High)
- Health recommendations
- Web-based interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- ML: Scikit-learn
- Cloud: Google Cloud Platform

## Disclaimer
This system is for educational purposes only and does not replace professional medical diagnosis.
